Free/open-source software for Mac OS X

  • Back to my page: http://www.mattmollison.com
  • Bold titles denote good stuff!
  • NB: I try to keep this list updated, but unfortunately it may be sorely out of date.
  • Here's another good list, if also a bit out of date: http://www.opensourcemac.org
  • NB for installing applications: After you mount a disk image, be sure to copy the application from the disk image to your Applications folder before running it. It is a good idea to eject the disk image and delete the .dmg file once you have done this.

Applications/Utilities

  • Quicksilver - application/file launcher, etc. (many plugins for other actions; check the plugins page) http://qsapp.com
    • User manual: http://mysite.verizon.net/hmelman/Quicksilver.pdf
    • Cheat sheet: http://mysite.verizon.net/hmelman/QSRef.pdf
  • nvAlt - simple note taking software http://brettterpstra.com/project/nvalt/
    • Sync wirelessly with iOS devices using a SimpleNote account http://simplenoteapp.com/
    • Notational Velocity is the original http://notational.net/
  • Wunderlist - to-do list that synchronizes across computers and iOS/Android devices https://www.wunderlist.com (also on Mac App Store)
  • Evernote - keep track of notes, syncs across computers and iOS devices http://www.evernote.com (also on Mac App Store)
  • SelfControl - stop procrastinating by preventing access to particular websites/domains for a set period of time http://selfcontrolapp.com/
  • Flux (f.lux) - Adjust computer display hue for night time use https://justgetflux.com
  • Itsycal - menu bar calendar, shows date and events http://www.mowglii.com/itsycal/
  • Day-O - menu bar clock replacement with a calendar http://shauninman.com/archive/2011/10/20/day_o_mac_menu_bar_clock
  • Pester - simple alarm clock and timer http://sabi.net/nriley/software/
  • Stash - personal finance manager https://github.com/ppearson/Stash/wiki/
  • MassReplaceIt - replace a string in multiple files/filenames http://www.hexmonkeysoftware.com
  • Homebrew - package manager http://brew.sh
  • Tofu - display text in an easier-to-read format http://amarsagoo.info/tofu/index.shtml

Science

  • Anaconda - full Python distribution for scientific computing http://continuum.io/downloads
  • Scipy Superpack - Python scientific stack (Numpy, Scipy, Matplotlib, ipython, pandas, Statsmodels, Scikit-Learn, and PyMC), installed via homebrew http://fonnesbeck.github.io/ScipySuperpack/
  • PyEPL - library for coding psychology experiments in Python http://pyepl.sourceforge.net
  • PsychoPy - good experiment complement to PyEPL (e.g., plays video) http://www.psychopy.org/
  • R - statistical analysis software http://www.r-project.org
    • Emacs Speaks Statistics - Code and run R in Emacs http://ess.r-project.org
  • Octave - For doing numerical computations, like Matlab http://www.gnu.org/software/octave/

System enhancement/utilities

  • Dropbox - online file synchronization that looks and acts like a folder in the Finder http://www.dropbox.com/referrals/NTM2MDU0MTk (yes, that is my referral URL :)
  • Menu Meters - system stats in menubar http://www.ragingmenace.com/software/menumeters/index.html
  • OnyX - system maintenance, turn on hidden features http://www.titanium.free.fr/pgs/english/apps.html
  • AppleJack - troubleshooting, system repair/maintenance http://sourceforge.net/projects/applejack
  • Growl - system notifications (not all versions are free, but supposedly all source code is downloadable) http://growl.info
  • The Unarchiver - decompress a variety of formats http://unarchiver.c3.cx (also on Mac App Store)
  • ClamXav - antivirus software http://www.clamxav.com
  • Omni DiskSweeper - show size of all directories and files in a browseable hierarchy (like a GUI for 'du -ah' at the command line) http://www.omnigroup.com/applications/omnidisksweeper/
  • ControlPlane - automatically configure settings based on your location/context http://www.controlplaneapp.com
  • VUWER - Track a stolen laptop http://stumpy.vuse.vanderbilt.edu/VUWER.htm
  • Prey - Track a stolen laptop (iOS/Linux/Windows devices, too) https://preyproject.com
  • iAlertU - alarm/notification security for Mac laptops http://www.macupdate.com/info.php/id/21282/ialertu
  • WordService - powerful text manipulation via the services menu http://www.devon-technologies.com/download/index.html
  • Dock Dodger - prevent a running application from appearing in the Dock http://www.macupdate.com/app/mac/24427/dock-dodger
  • QuickSynergy - share mouse and keyboard between multiple computers http://code.google.com/p/quicksynergy/
  • Mouse Locator - easily find cursor on the screen (good for large screens) http://www.2point5fish.com
  • BetterTouchTool - add more gestures to the Apple Magic Mouse http://www.macupdate.com/info.php/id/32953/bettertouchtool
  • fseventer - graphically observe filesystem changes http://www.fernlightning.com/doku.php?id=software:fseventer:start
  • GeekTool - show system logs, command-line output on desktop http://projects.tynsoe.org/en/geektool
  • Monolingual - remove extra language localization files to free up hard disk space http://monolingual.sourceforge.net

Data backup

  • Unison - backup files and/or keep files on multiple computers in sync http://www.cis.upenn.edu/~bcpierce/unison/
  • SuperDuper - hard drive backup/cloner http://www.shirt-pocket.com/SuperDuper

Internet/Network

  • Two-way sync between Google Calendar and iCal: https://support.google.com/calendar/answer/99358
    • Add Google-based push calendars to an iOS device for a good syncing setup https://support.google.com/mail/answer/3008051
    • But be sure to check out Apple's iCloud.
  • Safari Extensions https://extensions.apple.com/
  • Cyberduck - FTP client http://cyberduck.ch
  • JollysFastVNC - VNC client (free trial) http://www.jinx.de/JollysFastVNC.html

  • KisMAC - find/analyze wireless networks http://kismac-ng.org
  • iStumbler - find/analyze wireless networks http://www.istumbler.net/
  • Tor - anonymous Internet communication system http://www.torproject.org

Media: receive

Media: create - text/word processing

  • Google Docs - Google's web-based word processor, spreadsheets, and presentation manager (similar to Microsoft Office) http://docs.google.com (requires a free Google account)
  • LibreOffice - word processor, spreadsheet, presentations (similar to Microsoft Office) http://www.libreoffice.org/
  • Sublime Text - powerful text editor (free trial) http://www.sublimetext.com
  • KompoZer - WYSIWYG/HTML webpage-authoring (similar to Adobe Front Page or Dreamweaver) http://www.kompozer.net
  • Skim - PDF reader and note-taker (with excellent TeX document preview) http://skim-app.sourceforge.net
  • Aquamacs - Emacs for OS X http://aquamacs.org
  • Carbon Emacs - another Emacs app http://homepage.mac.com/zenitani/emacs-e.html
  • Muse Mode for Emacs - Emacs-based personal wiki http://www.mwolson.org/projects/MuseMode.html
  • VoodooPad Lite - personal wiki http://flyingmeat.com/voodoopad/voodoopadlite.html
  • MacTeX - LaTeX typesetting tools (installs TeX Live and several GUI utilities) http://www.tug.org/mactex
  • TeXShop - LaTeX GUI, TeX document preview http://www.uoregon.edu/~koch/texshop
  • BibDesk - BibTeX management, PDF organizer, integrates well with Skim http://bibdesk.sourceforge.net

  • Celtx - comprehensive screenplay writing package http://www.celtx.com
